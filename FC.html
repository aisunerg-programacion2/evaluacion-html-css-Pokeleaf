Etiquetas de Formularios y Carga de Datos

<form>
Se utiliza para iniciar un contenedor para varios tipos de metodos de entrada.

<input>
Es el tipo de formulario mas usado, mostrandose de maneras distintas segun el atributo type.
Por ejemplo <input type:"text"> crea un campo de una sola linea para introducir texto.





<label>
Sirve para etiquetar e identificar varios tipos de elementos. Como en el ejemplo anterior. "Nombre" y "Apellido"

<input type:"radio"> Crea un tipo de seleccion en el cual el usuario solo puede elegir una de las opciones.




<input type:"checkbox"> Define una seleccion de tipo casillas, en las cuales el usuario puede elegir desde varias a ninguna.




<input type:"submit">
Crea un boton para enviar y procesar lo que el usuario introdujo previamente en otra area.






<select>
Define un menú desplegable.

<option>
Define opciones para el menú desplegable.

<optgroup>
Define un grupo de opciones relacionadas.

<textarea>
Define un campo de texto de multiples lineas.

<button>
Define un botón.

<fieldset>
Define un grupo de datos relacionados.
<legend>
Define un titulo para el grupo de <fieldset>

<datalist>
Define una lista con opciones predefinidas para otros elementos <input>

<output>
Representa el resultado de algun calculo, como puede ser el de un script.

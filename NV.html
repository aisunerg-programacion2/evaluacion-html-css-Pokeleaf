Etiquetas de Numeración y Viñetas

Numeración

Para listas enumeradas se utiliza <ol> para definirla como tal, luego se añaden los elementos con <li>.

    Ejemplo de Numeración:
  1. Primer Objeto
  2. Segundo Objeto
  3. Tercer Objeto

Viñetas

Para listas sin numeración se utiliza <ul> para definir la lista y luego se añaden los elementos con <li>.

    Ejemplo de Viñetas:
  • Primer Objeto
  • Segundo Objeto
  • Tercer Objeto

Lista Descriptiva

Las listas descriptivas contienen terminos junto a sus descripciones.
Se define este tipo de lista con <dl>.
Y con <dt> se añade el termino a describir mientras que con <dd> se añade la descripcion de cada termino.

Ejemplo de lista Descriptiva:
Termino 1
- Descrición del Primer Termino
Termino 2
- Descripción del Segundo Termino
Termino 3
- Descripción del Tercer Termino

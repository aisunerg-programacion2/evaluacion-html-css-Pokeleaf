Etiquetas de Tablas

Etiqueta Función
<table> Define una tabla de HTML
<tr> Define una fila de la tabla
<th> Define el encabezado de la tabla
<border> Define un borde para la tabla
<collapse> Define si los bordes multiples pueden combinarse
<padding> Define el espacio entre el contenido y los bordes de la tabla
<border-spacing> Define el espeacio entre las celdas
<colspan> Define cuando una celda ocupa mas de una columna
<rowspan> Define cuando una celda ocupa mas de una fila
<caption> Define un titulo a la tabla
<colgroup> Define un grupo de columnas para opciones de formato
<col> Define la propiedades de cada columna en un grupo <colgroup>
<thead> Agrupa los encabezados de una tabla
<tbody> Agrupa el cuerpo de una tabla
<tfoot> Agrupa el pie de una tabla
